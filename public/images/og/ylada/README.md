# Imagens OG — Links YLADA (/l/[slug])

Imagens de prévia para links de quizzes compartilhados no WhatsApp.

## Pasta única compartilhada

Médico, nutricionista, fitness etc. usam a **mesma imagem por tema**.
Ex: Calculadora de IMC → `peso-gordura.jpg`, independente da área.

Mapeamento tema → arquivo em `src/lib/ylada-og-tema-imagem.ts`.

## Como começar

1. Substitua os placeholders pelas imagens finais (1200×630 px)
2. Consulte `docs/IMAGENS-OG-QUIZZES-YLADA.md` para a lista completa
