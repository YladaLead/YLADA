# Fontes das OG genéricas Pro Líderes

Coloca aqui os PNG de trabalho (nomes fixos):

- `og-default-recrutamento.source.png` — linha negócio / recrutamento  
- `og-default-saude.source.png` — linha saúde / vendas wellness  

Depois gera os JPEG 1200×630 servidos ao público:

```bash
npm run og:build-pro-lideres-defaults
```

Isto grava `../og-default-recrutamento.jpg` e `../og-default-saude.jpg` com ajustes de cor (fundo mais cinzento no recrutamento; saúde mais verde/viva).
