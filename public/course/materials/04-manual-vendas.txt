# COMO VENDER MELHOR - HERBALEAD

## 💰 COMO IDENTIFICAR SEU CLIENTE

### Cliente Frio (Primeira vez)
**Como é:**
- Não te conhece
- Não sabe dos seus produtos
- Precisa de educação

**Como falar:**
```
"Oi! Vi que você fez nosso quiz.
Que legal saber que você se preocupa com isso!
Posso te enviar algumas dicas gratuitas?
Sem compromisso, só para ajudar 😊"
```

### Cliente Morno (Já conhece)
**Como é:**
- Já viu seus produtos
- Tem interesse
- Está pesquisando

**Como falar:**
```
"Oi! Vi que você tem interesse em [produto].
Tenho clientes com resultados incríveis!
Quer que eu te mostre como eles conseguiram?
Posso te contar em 15 minutos 😊"
```

### Cliente Quente (Quer comprar)
**Como é:**
- Já decidiu
- Só quer saber como
- Pronto para comprar

**Como falar:**
```
"Oi! Vi que você está interessado em [produto].
Que tal eu te explicar como funciona?
Posso te dar um desconto especial hoje! 😊"
```

## 📞 COMO FAZER FOLLOW-UP

### Dia 1: Primeira Mensagem
**Objetivo:** Confirmar que recebeu

**Template:**
```
"Oi [Nome]! Recebi seu resultado do quiz.
Que interessante saber que você [resultado]!
Posso te ajudar com algumas dicas?
Sem compromisso, só para trocar uma ideia 😊"
```

### Dia 3: Oferecer Valor
**Objetivo:** Dar algo útil

**Template:**
```
"Oi [Nome]! Preparei algumas dicas para você.
Baseado no seu perfil, acredito que [dica].
Quer que eu te envie um material gratuito?
É só responder que eu mando agora! 😊"
```

### Dia 7: Convidar para Conversar
**Objetivo:** Marcar uma conversa

**Template:**
```
"Oi [Nome]! Última mensagem sobre isso.
Tenho uma oportunidade especial hoje para quem
realmente quer resolver [problema].
Quer conversar por 15 minutos?
Posso te dar um desconto exclusivo! 😊"
```

## 🎯 COMO RESPONDER OBJEÇÕES

### "Muito caro"
**Resposta:**
```
"Entendo sua preocupação com o preço.
Mas pense: quanto você gasta por mês com
coisas que não resolvem seu problema?
O [produto] custa menos que [comparação] e
resolve definitivamente. Vale a pena! 😊"
```

### "Preciso pensar"
**Resposta:**
```
"Claro! É uma decisão importante.
Mas me conta: o que te deixa em dúvida?
Posso esclarecer qualquer coisa agora.
Às vezes uma dúvida simples resolve tudo! 😊"
```

### "Não tenho tempo"
**Resposta:**
```
"Entendo! Todo mundo está correndo.
Mas o [produto] é justamente para quem não tem tempo!
Você toma [tempo] e pronto. Sem complicação.
Quer que eu te mostre como é simples? 😊"
```

### "Já uso outro produto"
**Resposta:**
```
"Que legal! É sempre bom cuidar da saúde.
Mas me conta: está funcionando como esperado?
Às vezes a gente precisa de algo mais específico.
Posso te explicar a diferença! 😊"
```

## 🚀 COMO FECHAR A VENDA

### Fechamento por Urgência
```
"Olha, tenho uma condição especial hoje que
não posso repetir. Se você decidir agora,
posso te dar [benefício especial].
Mas só até [prazo]. O que acha?"
```

### Fechamento por Benefício
```
"Imagina como seria sua vida se você
[benefício principal]? É isso que o [produto]
pode fazer por você. Vale a pena tentar?
Posso te ajudar a começar hoje!"
```
