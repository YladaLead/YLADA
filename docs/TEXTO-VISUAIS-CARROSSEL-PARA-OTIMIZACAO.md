# Texto dos visuais do carrossel — Para otimização com ChatGPT

**Objetivo:** Documentar todo o texto que aparece DENTRO das imagens/mockups do carrossel, para você levar ao ChatGPT e pedir melhorias de comunicação visual e copy.

---

## Problemas identificados

1. **Slide 11 (fechamento):** Precisa ser mais forte — mostrar vantagens, promessa ("seu trabalho fica mais leve"), benefícios. Hoje só tem uma frase.
2. **"Bloqueio":** Palavra negativa. Aparece em slides 5, 6, 7, 9. Alternativas: "desafio", "próximo passo", "foco", "prioridade".
3. **Visuais:** Podem transmitir melhor — cada imagem precisa ser revisada.

## Proposta: novo slide 11 (fechamento com vantagens)

O usuário quer que o slide 11 seja mais de "fechamento" — mostrando a promessa, as vantagens, o ganho. Exemplos:

- Seu trabalho fica mais leve
- Sua comunicação fica mais fácil
- Você para de convencer
- Pessoas chegam prontas
- Sem correr atrás

**Sugestão de estrutura para o slide 11:**
- Título: "Seu trabalho fica mais leve."
- Descrição: "Comunicação mais fácil. Pessoas prontas. Sem convencer."
- **Visual:** Lista de vantagens ou frase forte + benefícios

**Texto para o visual (slide 11):**
- Opção A: "Você só precisa perguntar. O Noel faz o resto."
- Opção B: Lista de 3–4 benefícios:
  - Comunicação mais fácil
  - Pessoas chegam prontas
  - Trabalho mais leve
  - Sem correr atrás
