REGRAS DE CONVERSÃO:
1. "não informado" ou vazio → null (não string vazia)
2. Datas: "DD/MM/YYYY" → "YYYY-MM-DD"
3. Pesos: converter lbs para kg (dividir por 2.20462), sempre número
4. Altura: "1,66 m" → 1.66 (número)
5. Horários: "7h" → "07:00", "23h" → "23:00"
6. Arrays: "Item1, Item2" → ["Item1", "Item2"]
7. Booleanos: "sim" → true, "não" → false/null
8. Sexo: "Feminino" → "feminino", "Masculino" → "masculino"
9. Funcionamento intestinal: "Diário" → "diario"
10. Qualidade sono: "Boa" → "bom", "Ótima" → "otimo"
11. Objetivo: "Emagrecimento" → "emagrecimento"
12. Água: extrair número de litros (ex: "3 litros/dia" → 3)
13. Medicamentos: array de objetos {name, dose} ou []
14. Evolução peso: array de objetos {date, weight, observation}
15. Reavaliações: array de objetos com todos os dados

IMPORTANTE:
- Sempre gere JSON válido
- Use null para campos não informados (não strings vazias)
- Arrays vazios = [] (não null)
- Números são números (não strings)
- Datas no formato YYYY-MM-DD

Agora converta a ficha de cliente que vou enviar para este formato JSON.
