# Brief: Quizzes com Gatilhos para Conversão

**Objetivo:** Criar tópicos de quiz que levem o usuário a **entrar em contato com o profissional** (nutricionista, psicólogo, dentista, etc.).

---

## Contexto

O YLADA é uma plataforma que gera links inteligentes (quizzes, calculadoras, diagnósticos) para profissionais de saúde e bem-estar. Cada link serve para:
1. **Engajar** o lead com conteúdo útil
2. **Qualificar** com base nas respostas
3. **Converter** — fazer a pessoa querer falar com o profissional

O quiz precisa ter **gatilhos mentais** que criem urgência, curiosidade ou desejo de resolver o problema com ajuda especializada.

---

## O que precisamos de você

Para **cada profissão/segmento**, crie uma lista de **tópicos de quiz** com:

| Campo | Descrição |
|-------|-----------|
| **Tópico** | Nome do quiz (ex: "Descubra seu nível de energia") |
| **Dor principal** | Qual dor o quiz aborda (ex: "Falta de energia") |
| **Objetivo principal** | O que o usuário quer alcançar (ex: "Melhorar disposição") |
| **Gatilho mental** | Qual gatilho usar (curiosidade, urgência, validação, medo de perder, etc.) |
| **Copy do resultado** | Frase que aparece no final e leva a agendar/contatar (ex: "Seu perfil indica que você pode se beneficiar de um acompanhamento personalizado. Agende uma conversa para descobrir o próximo passo.") |
| **CTA sugerido** | Texto do botão (ex: "Quero falar com um especialista", "Agendar avaliação") |

---

## Segmentos / Profissões

Use os segmentos abaixo como base. Cada um tem suas dores e objetivos já mapeados — os quizzes devem falar direto com isso.

### Nutrição
Dores: dificuldade de emagrecer, intestino preso, compulsão, inchaço, falta de energia, comer por ansiedade, etc.  
Objetivos: emagrecer com saúde, melhorar digestão, regular intestino, controlar fome emocional, etc.

### Estética
Dores: flacidez, celulite, manchas, rugas, gordura localizada, pele cansada, olheiras, etc.  
Objetivos: melhorar aparência da pele, rejuvenescer, diminuir celulite, melhorar contorno corporal, etc.

### Fitness
Dores: falta de motivação, cansaço no treino, baixa performance, dificuldade de ganhar massa, lesões, etc.  
Objetivos: melhorar condicionamento, ganhar massa, perder gordura, manter rotina de treinos, etc.

### Odontologia
Dores: mau hálito, sensibilidade, sangramento na gengiva, medo de dentista, dentes amarelados, etc.  
Objetivos: melhorar saúde bucal, clarear dentes, prevenir cáries, melhorar higiene oral, etc.

### Psicologia
Dores: ansiedade, estresse, dificuldade de concentração, problemas de relacionamento, pensamentos repetitivos, dificuldade de dormir, etc.  
Objetivos: reduzir ansiedade, equilíbrio emocional, melhorar foco, autoconhecimento, tranquilidade mental, etc.

### Médicos (bem-estar geral)
Dores: falta de energia, sedentarismo, alimentação desregulada, sono ruim, ganho de peso, etc.  
Objetivos: qualidade de vida, hábitos saudáveis, melhorar energia, reduzir estresse, etc.

### Vendedores de nutracêuticos/suplementos
Dores: cansaço, falta de concentração, inchaço, digestão ruim, retenção, dificuldade de emagrecer, etc.  
Objetivos: aumentar energia, melhorar foco, melhorar digestão, desinchar, melhorar metabolismo, etc.

## Formato de entrega

Use este template para cada tópico:

```
## [NOME DO SEGMENTO]

### Quiz 1
- **Tópico:** [nome do quiz]
- **Dor:** [dor que aborda]
- **Objetivo:** [objetivo do usuário]
- **Gatilho mental:** [curiosidade / urgência / validação / medo de perder / escassez / autoridade / etc.]
- **Copy do resultado:** [2-3 frases que aparecem no final e levam a contatar]
- **CTA:** [texto do botão]

### Quiz 2
...
```

## Exemplo completo (Nutrição)

### Quiz 1
- **Tópico:** Descubra seu nível de energia
- **Dor:** Falta de energia
- **Objetivo:** Melhorar energia diária
- **Gatilho mental:** Curiosidade + validação (a pessoa quer saber "o que está errado comigo")
- **Copy do resultado:** "Seu perfil indica que sua energia pode estar sendo drenada por hábitos que você nem percebe. Um nutricionista pode identificar os pontos exatos e montar um plano para você recuperar a disposição em poucas semanas."
- **CTA:** Quero melhorar minha energia

### Quiz 2
- **Tópico:** O que pode estar travando seu emagrecimento
- **Dor:** Dificuldade de emagrecer
- **Objetivo:** Emagrecer com saúde
- **Gatilho mental:** Urgência + medo de perder (o quiz revela "bloqueadores" que a pessoa não conhece)
- **Copy do resultado:** "Existem fatores invisíveis que podem estar impedindo seus resultados. Uma avaliação personalizada mostra exatamente o que ajustar para você começar a ver diferença."
- **CTA:** Quero uma avaliação personalizada

---

## Lembrete importante

**O propósito de cada quiz é que o usuário queira falar com o profissional.**  
O resultado não deve "fechar" o assunto — deve abrir a porta para uma conversa. Frases como "agende uma avaliação", "fale com um especialista", "descubra o próximo passo" reforçam que há algo a ser feito *com ajuda*.
