# Biblioteca Universal de Quizzes — YLADA

Organização principal: **por SEGMENTO do sistema YLADA**. Cada segmento pode ser utilizado por diferentes profissionais (ex.: estética → esteticistas, dermatologistas, clínicas; fitness → personal trainers, academias).

**Objetivo:** gerar diagnóstico inicial que desperta consciência e leva o usuário a falar com um profissional.

**Estrutura padrão de cada quiz:**
- Nome do quiz
- Dor principal
- Objetivo do usuário
- Gatilho mental
- Copy do resultado
- CTA
